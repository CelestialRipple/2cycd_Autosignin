# 2cycd_Autosignin
二次元虫洞自动签到，青龙面板支持

### 使用方法
论坛官网：http://www.2cycd.com
每天签到领贡献～

推荐使用青龙面板
使用浏览器登录之后, F12-网络选项卡-捉包把home.php/plugin.php请求里面的Cookie的值填到环境变量变量cycd_cookie里, 多账号换行或&或@隔开

或者自行修改main函数中cookie变量
